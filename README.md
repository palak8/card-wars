# Card War
Old popular card game, now on python, played as a simulation.


### Tech
Python3 (Jupyter Notebook)

### Run
- Open ```War Cards.ipynb``` on Jupyter Notebook (Python3 kernel) OR
- Directly run ```War Cards.py``` (Python3)
